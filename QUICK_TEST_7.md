## How to Test Responsive Design

### Option 1: Browser DevTools
1. Press F12 to open DevTools
2. Click "Toggle device toolbar" (or Ctrl+Shift+M)
3. Select device sizes:
   - Desktop: 1024px+
   - Tablet: 768px (iPad)
   - Mobile: 375px (iPhone)

### Option 2: Manual Resize
1. Resize browser window manually
2. Test at different widths:
   - Desktop: Full window
   - Tablet: ~768px width
   - Mobile: ~375px width
